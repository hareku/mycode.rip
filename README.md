## Development

### Using local Node.js

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000 after a few seconds later
$ yarn run dev
```

### Using Docker

``` bash
# install dependencies
$ docker-compose run --rm node yarn install

# serve with hot reload at localhost:3000 after a few seconds later
$ docker-compose up
```
