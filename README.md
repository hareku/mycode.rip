## Build Setup

``` bash
# install dependencies
$ npm install # Or yarn install

# serve with hot reload at localhost:3000
$ cross-env CTF_SPACE_ID=xxxxx CTF_CDA_ACCESS_TOKEN=xxxxx npm run dev

# generate static project
$ npm run generate
```
