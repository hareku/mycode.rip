## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ cross-env CTF_SPACE_ID=xxxxx CTF_CDA_ACCESS_TOKEN=xxxxx yarn run dev

# generate static project
$ yarn run generate
```
